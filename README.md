AI-Powered Intelligent Assessment Monitoring System

We propose an AI-powered assessment monitoring system that combines computer vision, a lightweight Large Language Model (LLM), and Retrieval-Augmented Generation (RAG) to assist invigilators during online examinations. The system monitors observable events such as prolonged gaze deviation, face absence, multiple faces, mobile phone presence, tab switching, and fullscreen exit. These events are converted into structured data and analyzed by an LLM using relevant examination rules retrieved through RAG. The system generates a risk level and an explainable alert for the invigilator. The final product is a real-time monitoring platform with a student-side assessment interface and an invigilator dashboard that prioritizes suspicious activities for human review while supporting efficient and privacy-aware examination monitoring.
